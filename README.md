# Open Pages

Open Pages é um blog estático simples sobre livros, desenvolvido com HTML e CSS puro.

## Sobre o projeto

O site reúne reflexões pessoais sobre livros em uma experiência legível e direta. A proposta técnica é manter a aplicação simples, sem frameworks, sem JavaScript e sem etapa de build, usando apenas arquivos estáticos que podem ser abertos localmente ou hospedados facilmente.

## Tecnologias utilizadas

- HTML5
- CSS3
- Estrutura semântica com `header`, `nav`, `main` e `footer`
- Layout responsivo
- Imagens locais em PNG
- Preparação para GitHub Pages

## Como executar localmente

Clone o repositório e acesse a pasta do projeto:

```bash
git clone https://github.com/fscyuri/open-pages.git
cd open-pages
```

Inicie um servidor local:

```bash
python3 -m http.server 8000
```

Depois abra no navegador:

```text
http://localhost:8000/index.html
```

Também é possível abrir o arquivo `index.html` diretamente no navegador, já que o projeto não depende de backend ou build.

## Objetivos técnicos

- Praticar HTML semântico.
- Criar navegação consistente entre múltiplas páginas.
- Organizar um site estático com arquivos simples.
- Trabalhar responsividade com CSS.
- Usar imagens locais como parte do conteúdo visual.
- Preparar o projeto para publicação futura no GitHub Pages.
